# Scribe

Install scribe
```bash
$ 
```

## Commands

```bash
$ scribe --set-password <value>
$ scribe --set-username <value>
```

```bash
$ scribe --verify
```

```bash
$ scribe --search
```

##--search

|Parameter|value|
|:-|:-|
|-i  |Id|
|-l  |Title|
|-e  |English|
|-c  |Score|
|-t  |Type|
|-p  |Episodes|
|-y  |Synonyms|
|-u  |Status|
|-r  |Start date|
|-f  |End date|
|-d  |Synopsis|
|-g  |Image|

----------

|Parameter|argument|description|
|:-|:-|
|-w  |**int**| How many results we should show. |
|-n  |**int**| The length of the synopsis.|
